# Callouts

从 v0.14.0 开始，Obsidian 支持 Callouts（标注块），灵感来自 Microsoft Docs "alert" 语法。

现在可以不再使用 admonition：<https://github.com/valentine195/obsidian-admonition>

使用参考：<https://help.obsidian.md/How+to/Use+callouts>
