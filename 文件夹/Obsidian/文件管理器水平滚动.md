# 文件管理器水平滚动

<https://forum.obsidian.md/t/deep-trees-overflow-the-file-explorer/45315>

```css
.nav-files-container {
  overflow-x: scroll;
}
.nav-file-title-content, .nav-folder-title-content {
  overflow: visible;
}
.nav-file-title, .nav-folder-title {
  min-width: 100%;
  width: max-content;
}
```

将 `.css` 放入 snippets 文件夹并开启：
